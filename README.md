# CMPSC132-Final-Project-Tic-Tac-Toe-Game
Alessio Rella's CMPSC 132 Final Project:
This is a 2 Player Tic Tac Toe game that is played in the terminal. Players take turns playing a symbol in a 3x3 grid. The game ends when one player connects 3 in a row or the board is filled without a winner (draw).

Extra Features:
- Adjustable symbols
- Scoreboard
- Reversible turn order

DATA STRUCTURES USEDl
- Dictionaries:
     - Board state
- Lists:
     - Rows, columns, and diagonals
     - Players

TO PLAY:
Run tictactoe.py in a python editor. The game will start playing in the interactive terminal. Follow the given prompts to play.
The gameboard is as follows: 
```
    1   2   3

A     |   |  
   -----------
B     |   |  
   -----------
C     |   |  
```
On your turn, enter a valid letter (row) and number (column) in the same input to play.
